🚀 快速上手 新用户必读

只需 5 分钟，从零开始成为 NFLSHC Chat 达人。

第一步：注册账号

  1. 访问 注册页面。
  2. 填写用户名、邮箱（用于找回密码）、密码（至少6位）。
  3. 点击注册，系统会将你的账号信息保存到 GitHub Issues 中。
  4. 注册成功后，会自动跳转到登录页面。
💡 提示：用户名建议使用英文或数字，方便他人 @ 你。

第二步：登录与进入聊天

  1. 在 登录页 输入用户名和密码。
  2. 管理员（huangzhiyuan）登录后会进入管理后台。
  3. 普通用户登录后自动进入 聊天主界面。
🔐 忘记密码？登录页点击「忘记密码」可邮件找回。

第三步：开始第一次聊天

  1. 在聊天主界面左侧，点击「+ 创建聊天室」。
  2. 选择「群聊」，输入名称（如“高一1班”），选择「公开」或「需验证码」。
  3. 创建成功后，点击进入聊天室。
  4. 在底部输入框输入文字，按 Enter 发送。
  5. 试试 Markdown：输入 **粗体** 或 # 标题 看看效果。
📝 小技巧：Shift + Enter 换行，Enter 直接发送。

第四步：认识界面元素

  • 左侧边栏：所有聊天室列表，点击切换。
  • 顶部导航：个人资料、统计、导出、收藏、搜索、AI助手、公告等入口。
  • 消息区：显示聊天记录，支持 Markdown 渲染。
  • 输入区：发送消息、引用回复、语音录音。

第五步：探索更多功能

  • 👤 点击「个人资料」完善头像和简介。
  • 👥 在「好友」页面添加好友，快速私聊。
  • ⭐ 鼠标悬停消息，点击 ⭐ 收藏重要内容。
  • 🔍 使用「搜索」快速查找历史消息。
  • 🤖 在群聊中输入 @AI 你好 体验 AI 助手。

✨ 核心功能

💬 智能对话思知 AI / Agnes AI / 智谱 GLM
🎨 AI 绘图Agnes Image 2.1 Flash / 智谱 Cogview-3-Flash
🎬 AI 视频生成Agnes Video V2.0 / 智谱 CogVideoX-Flash
👁️ 视觉理解智谱 GLM-4.6V-Flash / GLM-4.1V-Thinking-Flash

🛠️ 技术架构

  • 前端框架：纯原生 HTML/CSS/JavaScript，无任何第三方框架依赖
  • 对话引擎：支持思知 AI（免费）、Agnes AI（多模态）、智谱 GLM-4.7-Flash / GLM-4-Flash-250414
  • 绘图引擎：Agnes Image 2.1 Flash（图生图）、智谱 Cogview-3-Flash（文生图）
  • 视频引擎：Agnes Video V2.0（异步）、智谱 CogVideoX-Flash（异步）
  • 视觉引擎：智谱 GLM-4.6V-Flash（图片/视频/文件理解）、GLM-4.1V-Thinking-Flash（思维链推理）
  • 数据存储：浏览器 localStorage + GitHub Issues 云端同步（对话、角色、任务）
  • 语音输入：Web Speech API（Chrome/Edge 支持）
💡 HZYAI 的所有对话记录、角色数据和定时任务均可通过 GitHub Issues 跨设备同步。

🧠 模型选择指南

  • 思知 AI：完全免费，无限制，适合日常对话和简单问答。
  • Agnes 2.0 Flash：多模态模型，支持工具调用、编码、图像理解。
  • 智谱 GLM-4.7-Flash：30B 级 SOTA 模型，强化编码能力与长程规划。
  • 智谱 GLM-4-Flash-250414：免费文本模型，128K 上下文，适合长文本处理。
  • 绘图模型：Agnes Image 2.1 Flash（图生图）、智谱 Cogview-3-Flash（文生图）。
  • 视频模型：Agnes Video V2.0（异步）、智谱 CogVideoX-Flash（异步）。
  • 视觉模型：GLM-4.6V-Flash（图片/视频/文件）、GLM-4.1V-Thinking-Flash（思维链推理）、GLM-4V-Flash（图片理解）。

💡 使用技巧

  • 绘图技巧：描述越详细，生成效果越好。可上传参考图实现图生图。
  • 视频生成：视频生成需要约 1-3 分钟，请耐心等待，支持 3-18 秒时长。
  • 虚拟角色：创建角色后，在对话中选择角色，AI 会以该角色的身份和语气回应。
  • 定时任务：设置后 AI 会在指定时间自动推送新闻、天气或提醒。
  • 记忆系统：对话结束后 AI 会自动总结并写入角色记忆，下次对话可读取。
🎯 提示：在群聊中输入 @AI 或 @机器人 即可触发 HZYAI 智能助手。

❓ 常见问题

高频问题集中解答，如果这里没有你的问题，请联系管理员。

🔑 账号相关

  • 问：忘记密码怎么办？
    答：登录页面点击「忘记密码？」，输入用户名和注册邮箱，系统会将密码发送到您的邮箱（每月限 200 次，请勿滥用）。
  • 问：账号被封禁了怎么办？
    答：请联系管理员 huangzhiyuan 申诉解封。
  • 问：如何修改密码？
    答：进入「个人资料」页，点击「编辑个人资料」，输入新密码即可。
  • 问：如何设置自定义头像？
    答：在「个人资料」编辑页，输入头像图片的 URL 链接即可（支持图床链接）。
  • 问：为什么注册时提示用户名已存在？
    答：用户名在 GitHub Issues 中全局唯一，请更换用户名重试。

💬 聊天相关

  • 问：消息发送失败怎么办？
    答：检查网络连接，确认账号未被封禁，刷新页面后重试。如果频繁失败，可能触发了 GitHub API 速率限制。
  • 问：如何撤回消息？
    答：发送后 2 分钟内，鼠标悬停在自己的消息上，点击「撤回」按钮。
  • 问：如何收藏消息？
    答：鼠标悬停在消息上，点击右下角的 ⭐ 按钮即可收藏。
  • 问：如何引用回复别人的消息？
    答：鼠标悬停在任意消息上，点击「↩️」按钮即可引用回复。
  • 问：如何发送语音消息？
    答：点击输入框旁边的 🎤 按钮开始录音，最长 60 秒，录制完成后自动发送。
  • 问：Markdown 代码块能高亮吗？
    答：可以！代码块自动进行语法高亮，鼠标悬停显示「复制」按钮。

👥 群聊相关

  • 问：如何创建群聊？
    答：点击「+ 创建聊天室」，选择「群聊」，填写名称和加入方式。
  • 问：如何设置群聊验证码？
    答：创建群聊时选择「需验证码加入」，或在「群设置」中修改。
  • 问：如何成为群管理员？
    答：群主可以在「群设置」中添加管理员。
  • 问：如何置顶群聊消息？
    答：群主和管理员鼠标悬停在消息上，点击「📍」按钮即可置顶。
  • 问：如何退出群聊？
    答：在「群设置」中点击「退出群聊」。

👤 好友相关

  • 问：如何添加好友？
    答：进入「好友」页面，输入用户名点击「添加好友」即可。
  • 问：如何查看好友的等级？
    答：在好友列表中会显示好友的头像和等级徽章。
  • 问：如何与好友私聊？
    答：在好友列表中点击「💬 发消息」按钮。

🤖 AI 相关

  • 问：AI 不回复怎么办？
    答：确认在群聊中使用了 @AI 或 @机器人 触发词，检查网络连接。
  • 问：AI 支持哪些功能？
    答：对话、天气查询、翻译、笑话、计算、古诗查询等。
  • 问：HZYAI 是什么？
    答：HZYAI 是独立的 AI 对话工具，支持对话、绘图、视频生成，访问速度更快。

📊 数据相关

  • 问：聊天记录保存在哪里？
    答：所有数据保存在 GitHub Issues 中，完全透明，可审计。
  • 问：如何导出聊天记录？
    答：点击导航栏「📤 导出」，选择导出范围和格式（JSON 或 HTML）。
  • 问：数据安全吗？
    答：数据存储在 GitHub 服务器，但 Issue 是公开的，请勿发送敏感信息。

🛠️ 技术相关

  • 问：网站是免费的吗？
    答：是的，完全免费使用。
  • 问：支持移动端吗？
    答：支持响应式设计，手机和电脑都可以正常使用。
  • 问：为什么有时候会卡？
    答：由于基于 GitHub API，网络延迟或速率限制可能导致卡顿，请适当刷新。
  • 问：有桌面版或 App 吗？
    答：有 Windows 桌面版（v2.3.0）和 Android APK，可在公告或社区页面获取。
