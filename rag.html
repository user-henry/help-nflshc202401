📖 一、项目概述

👤 二、用户系统

2.1 注册

  • 用户需要填写用户名、邮箱、密码（至少6位）进行注册
  • 可以填写个人简介（选填）
  • 注册成功后自动跳转到登录页面
  • 数据存储：GitHub Issues（label: user）

2.2 登录

  • 输入用户名和密码登录
  • 管理员（huangzhiyuan）登录后跳转到管理后台（admin.html）
  • 普通用户登录后跳转到聊天页面（chat.html）
  • 支持忘记密码功能（通过邮箱找回，每月限200次）

2.3 个人资料（profile.html）

  • 查看自己的用户名、个人简介、参与的聊天室
  • 查看等级（Lv.）和经验值（XP），经验值由消息数量决定
  • 普通用户可查看注册时间，管理员可查看等级和统计
  • 编辑个人简介、修改密码
  • 设置自定义头像（通过头像图片URL）
  • 查看他人资料（点击消息中的用户名）
  • 管理员可通过个人资料页进入管理员中心
  • 支持分享个人名片（生成个人资料链接）

2.4 用户等级系统

  • 等级（Lv.）由经验值（XP）计算，XP = 消息数量
  • 等级公式：Lv. = floor(sqrt(XP / 10)) + 1
  • 等级展示：Lv.1-3 绿色，Lv.4-6 蓝色，Lv.7-9 紫色，Lv.10+ 金色
  • 等级徽章显示在消息发送者旁边和用户资料页
  • 个人数据仪表盘展示详细等级进度条和下一级所需经验

💬 三、聊天功能

3.1 聊天室列表

  • 左侧显示所有可用的聊天室
  • 群聊显示绿色标签，私聊显示红色标签
  • 需要验证码的群聊显示 🔐 图标

3.2 创建聊天室

  • 点击「+ 创建聊天室」按钮
  • 选择类型：群聊 或 私聊
  • 群聊可设置加入方式：公开（任何人可加入）或 需验证码加入
  • 验证码可自定义或随机生成
  • 私聊需要输入对方用户名
  • 提供「🤖 与 AI 小助手私聊」快捷按钮

3.3 加入群聊

  • 点击群聊名称即可加入
  • 如需验证码，输入正确验证码后加入

3.4 发送消息

  • 支持 Markdown 格式（粗体、斜体、链接、代码块、列表、表格、引用等）
  • 输入时下方显示实时预览效果
  • 支持 @用户名 提到特定用户
  • 支持 @所有人 通知全体成员（仅群聊）
  • 按 Enter 键发送，Shift+Enter 换行

3.5 消息撤回

  • 发送后 2 分钟内可以撤回
  • 鼠标悬停在自己的消息上，点击「撤回」按钮

3.6 消息收藏

  • 鼠标悬停在消息上，点击右下角 ⭐ 按钮即可收藏
  • 收藏的消息在「我的收藏」页面查看
  • 支持删除收藏
  • 数据存储：GitHub Issues（label: favorite）

3.7 消息搜索（search.html）

  • 支持按关键词搜索历史消息
  • 支持按聊天室筛选
  • 支持按发送者筛选
  • 支持按日期范围筛选
  • 搜索结果高亮显示关键词，分页加载（每页20条）

3.8 聊天记录导出（export.html）

  • 导出所有聊天记录（JSON格式）
  • 导出指定聊天室记录（JSON或HTML格式）
  • 按时间范围导出

3.9 消息引用回复

  • 鼠标悬停在任意消息上，点击「↩️」按钮进行引用回复
  • 被回复的消息以引用缩略形式显示在回复上方
  • 引用内容显示原消息发送者和内容摘要

3.10 语音消息

  • 点击输入框旁边的 🎤 按钮开始录音
  • 支持最长 60 秒录音
  • 录制完成后自动发送语音消息
  • 语音消息以音频播放器形式展示，点击可播放
  • 技术实现：MediaRecorder API，base64 编码存储

3.11 代码语法高亮

  • 代码块自动进行语法高亮（支持多种编程语言）
  • 鼠标悬停在代码块上显示「📋 复制」按钮
  • 点击复制按钮一键复制代码内容
  • 集成 highlight.js（atom-one-dark 主题）

👥 四、群聊管理

4.1 群设置（仅群主和管理员）

  • 编辑群公告
  • 修改加入方式（公开/需验证码）
  • 生成/修改验证码
  • 添加/移除群管理员
  • 退出群聊

4.2 群聊置顶消息

  • 群主和管理员可置顶重要消息
  • 鼠标悬停在消息上，点击「📍」按钮置顶（已置顶显示「📌」）
  • 置顶消息在聊天室顶部显示横幅，点击可跳转查看
  • 再次点击「📌」可取消置顶

4.3 群聊统计（stats.html）

  • 总消息数
  • 活跃成员数
  • 日均消息量
  • 活跃天数
  • 消息趋势图（最近30天）
  • 时段分布图（24小时）
  • 高频词汇云
  • 发言排行榜（Top 10）
  • 技术实现：Chart.js

👥 五、好友系统（friends.html）

5.1 添加好友

  • 在「好友」页面输入用户名即可添加好友
  • 系统自动验证用户是否存在
  • 不可添加自己或 AI 小助手为好友
  • 不可重复添加已添加的好友
  • 数据存储：GitHub Issues（label: friend）

5.2 好友列表

  • 显示所有已添加的好友
  • 显示好友头像、等级（Lv.）、个人简介
  • 支持搜索好友（按用户名筛选）
  • 点击好友名称可查看其个人资料

5.3 与好友聊天

  • 点击「💬 发消息」按钮直接进入私聊
  • 点击「❌ 删除」按钮可移除好友

📊 六、个人数据仪表盘（dashboard.html）

6.1 统计概览

  • 总消息数
  • 好友数
  • 参与的聊天室数
  • 收藏消息数

6.2 等级进度

  • 显示当前等级（Lv.）和称号
  • 进度条展示当前经验值距离下一级的进度
  • 经验值来源：消息数量

6.3 数据图表

  • 消息趋势图（最近14天）
  • 活跃时段分布图（24小时）
  • 聊天室消息分布

6.4 成就徽章

  • 根据消息数、好友数、收藏数等达成成就
  • 成就包括：初来乍到、话痨初级、话痨中级、话痨高级、超级话痨、社交新手、社交达人、人气之王等

👑 七、管理员功能

  • 查看所有用户列表

7.2 群聊管理

  • 查看所有聊天室（包括私聊）

  • 封禁/解封群聊

7.3 公告管理（notice.html）

  • 发布公告（支持 Markdown）
  • 支持编写/预览双标签页
  • 删除公告
  • 公告按时间倒序排列
  • 数据存储：GitHub Issues（label: notice）

7.4 建议审核（admin-suggestions.html）

  • 查看用户提交的建议
  • 更新建议状态：待审核、处理中、已完成、已驳回
  • 添加备注说明
  • 数据存储：GitHub Issues（label: suggestion）

❓ 十三、常见问题

账号相关

  • 忘记密码？ 登录页点击「忘记密码？」，输入用户名和注册邮箱，系统会将密码发送到您的邮箱。
  • 账号被封禁？ 联系管理员 huangzhiyuan 申诉解封。
  • 修改密码？ 进入「个人资料」页，点击「编辑个人资料」，输入新密码即可。
  • 自定义头像？ 在「个人资料」编辑页，输入头像图片的 URL 链接即可。

聊天相关

  • 消息发送失败？ 检查网络连接，确认账号未被封禁，刷新页面后重试。
  • 撤回消息？ 发送后 2 分钟内，鼠标悬停在自己的消息上，点击「撤回」按钮。
  • 引用回复？ 鼠标悬停在任意消息上，点击「↩️」按钮即可引用回复。
  • 语音消息？ 点击输入框旁边的 🎤 按钮开始录音，最长 60 秒。

群聊相关

  • 创建群聊？ 点击「+ 创建聊天室」，选择「群聊」，填写名称和加入方式。
  • 设置验证码？ 创建时选择「需验证码加入」，或在「群设置」中修改。
  • 成为管理员？ 群主可以在「群设置」中添加管理员。
  • 置顶消息？ 群主和管理员鼠标悬停在消息上，点击「📍」按钮即可置顶。

AI 相关

  • AI 不回复？ 确认在群聊中使用了 @AI 或 @机器人 触发词。
  • HZYAI 是什么？ 独立的 AI 对话工具，支持对话、绘图、视频生成。

数据相关

  • 数据保存在哪里？ 所有数据保存在 GitHub Issues 中，完全透明。
  • 导出聊天记录？ 点击导航栏「📤 导出」，选择导出范围和格式。

技术相关

  • 免费吗？ 完全免费。
  • 支持移动端？ 支持响应式设计。
  • 为什么卡？ 基于 GitHub API，网络延迟或速率限制可能导致卡顿。
  • 有桌面版？ 有 Windows 桌面版（v2.3.0）和 Android APK。
