1. 注册 GitHub 账号 第一步

如果你还没有 GitHub 账号，请按以下步骤注册：

  1. 访问 github.com
  2. 点击右上角 Sign up
  3. 填写用户名、邮箱、密码
  4. 完成邮箱验证
  5. 登录后进入 GitHub 首页
💡 用户名建议：使用英文或数字组合，例如 yourname 或 yourname-chat

2. 创建仓库 第二步

仓库（Repository）是存放你所有代码的地方。

  1. 点击 GitHub 右上角的 + 号，选择 New repository
  2. 填写 Repository name，例如 my-chat-app
  3. 选择 Public（公开仓库，免费）
  4. 勾选 Add a README file
  5. 点击 Create repository
💡 仓库名建议：使用 用户名.github.io 的格式（如 yourname.github.io），这样部署后可以直接通过 https://yourname.github.io 访问。

3. 获取 Personal Access Token 第三步

Token 是你的网站访问 GitHub 数据的钥匙。

  1. 点击右上角头像 → Settings
  2. 左侧菜单最下方 → Developer settings
  3. 点击 Personal access tokens → Tokens (classic)
  4. 点击 Generate new token → Generate new token (classic)
  5. 填写 Note：我的聊天网站
  6. 设置 Expiration：建议 No expiration（无过期）或 90 days
  7. 勾选权限：repo（全部勾选）
  8. 滚动到底部点击 Generate token
  9. ⚠️ 立即复制 Token！ 页面刷新后就不再显示
⚠️ 安全警告：Token 相当于密码，不要分享给任何人，也不要上传到公开仓库！
💡 Token 格式：类似 ghp_xxxxxxxxxxxxxxxxxxxx

4. 项目文件结构 第四步

你的网站需要以下核心文件：

📁 my-chat-app/ ├── 📄 index.html # 登录/首页 ├── 📄 chat.html # 聊天主页面（核心） ├── 📄 config.js # GitHub 配置（包含 Token） ├── 📄 style.css # 样式文件 └── 📄 README.md # 项目说明（可选）
💡 核心文件说明：
config.js — 存储你的 GitHub 用户名、仓库名、Token
chat.html — 所有聊天逻辑（发送消息、加载消息、AI 对话）
index.html — 用户登录入口

5. 核心代码详解 第五步

5.1 config.js — 配置文件

将以下代码复制到 config.js 中，替换成你的信息：

// config.js — 请替换成你的信息 window.CONFIG = { GITHUB_USERNAME: '你的GitHub用户名', GITHUB_REPO: '你的仓库名', GITHUB_TOKEN: '你的Personal Access Token' };

5.2 用户登录系统 (index.html 核心)

<!-- index.html 登录验证核心 --> <script> async function login() { const username = document.getElementById('username').value; const password = document.getElementById('password').value; // 从 GitHub Issues 读取用户数据 const url = `https://api.github.com/repos/${window.CONFIG.GITHUB_USERNAME}/${window.CONFIG.GITHUB_REPO}/issues?labels=user&state=open`; const res = await fetch(url, { headers: { 'Authorization': `token ${window.CONFIG.GITHUB_TOKEN}` } }); const issues = await res.json(); // 查找匹配的用户 for (const issue of issues) { const match = issue.body.match(/```json\n([\s\S]*?)\n```/); if (match) { const user = JSON.parse(match[1]); if (user.username === username && user.password === password) { sessionStorage.setItem('currentUser', JSON.stringify(user)); window.location.href = 'chat.html'; return; } } } alert('用户名或密码错误'); } </script>

5.3 发送消息 (chat.html 核心)

// chat.html — 发送消息到 GitHub Issues async function sendMessage() { const input = document.getElementById('messageInput'); const content = input.value.trim(); if (!content) return; const messageData = { id: `msg_${Date.now()}`, roomId: currentRoom.id, sender: currentUser.username, content: content, timestamp: new Date().toISOString() }; // 存储到 GitHub Issues const issueBody = `聊天消息\n\`\`\`json\n${JSON.stringify(messageData, null, 2)}\n\`\`\``; await fetch(`https://api.github.com/repos/${window.CONFIG.GITHUB_USERNAME}/${window.CONFIG.GITHUB_REPO}/issues`, { method: 'POST', headers: { 'Authorization': `token ${window.CONFIG.GITHUB_TOKEN}`, 'Content-Type': 'application/json' }, body: JSON.stringify({ title: `Message: ${messageData.id}`, body: issueBody, labels: ['chatmessage'] }) }); input.value = ''; loadMessages(); // 刷新消息列表 }

5.4 加载消息 (chat.html 核心)

// chat.html — 从 GitHub Issues 加载消息 async function loadMessages() { const url = `https://api.github.com/repos/${window.CONFIG.GITHUB_USERNAME}/${window.CONFIG.GITHUB_REPO}/issues?labels=chatmessage&state=open`; const res = await fetch(url, { headers: { 'Authorization': `token ${window.CONFIG.GITHUB_TOKEN}` } }); const issues = await res.json(); const messages = []; for (const issue of issues) { const match = issue.body.match(/```json\n([\s\S]*?)\n```/); if (match) { const msg = JSON.parse(match[1]); // 只显示当前聊天室的消息 if (msg.roomId === currentRoom.id) { messages.push(msg); } } } // 按时间排序 messages.sort((a, b) => new Date(a.timestamp) - new Date(b.timestamp)); renderMessages(messages); }

5.5 完整的 HTML 模板

<!DOCTYPE html> <html> <head> <meta charset="UTF-8"> <title>我的聊天网站</title> <script src="config.js"></script> </head> <body> <div id="app"> <div id="messages"></div> <div id="input-area"> <input type="text" id="messageInput" placeholder="输入消息..."> <button onclick="sendMessage()">发送</button> </div> </div> <script src="chat.js"></script> </body> </html>

6. 部署到 GitHub Pages 第六步

将你的代码推送到 GitHub，然后启用 Pages 服务。

  1. 创建文件：在仓库中点击 Add file → Create new file
  2. 上传代码：分别创建 index.html、chat.html、config.js
  3. 点击 Commit new file
  4. 进入仓库 Settings → Pages
  5. 在 Branch 下选择 main，点击 Save
  6. 等待 1-2 分钟，页面顶部会显示 https://你的用户名.github.io/你的仓库名/
✅ 部署成功！ 现在所有人都可以通过这个链接访问你的网站了。
💡 自动部署：每次推送到 main 分支，GitHub Pages 会自动更新网站。

8. 添加 AI 对话能力 第八步

使用免费的思知 AI 大模型，让你的聊天网站拥有智能对话能力。

8.1 AI 对话核心代码

// AI 对话函数 const AI_CONFIG = { apiUrl: 'https://api.sizhi.com/chat', appid: '你的思知 AI AppID' }; async function callAI(message) { const url = `${AI_CONFIG.apiUrl}?appid=${AI_CONFIG.appid}&spoken=${encodeURIComponent(message)}&memory=true`; const response = await fetch(url); const data = await response.json(); if (data.status === 0) { return data.data.info.text; } return 'AI 暂时无法回答，请稍后再试。'; } // 在聊天中调用 AI async function sendWithAI() { const input = document.getElementById('messageInput'); const content = input.value.trim(); if (!content) return; // 发送用户消息 sendMessage(content); // 调用 AI 回复 const reply = await callAI(content); sendMessage(reply, 'assistant'); }

8.2 获取思知 AI AppID

  1. 访问 api.sizhi.com
  2. 注册账号并登录
  3. 进入控制台，创建应用
  4. 获取 AppID

9. 语音输入（浏览器自带） 第九步

使用 Web Speech API 实现语音转文字，无需任何第三方服务。

// 浏览器自带语音识别 function startVoiceInput() { const recognition = new (window.SpeechRecognition || window.webkitSpeechRecognition)(); recognition.lang = 'zh-CN'; recognition.continuous = false; recognition.interimResults = true; recognition.onstart = () => { console.log('🎤 请说话...'); }; recognition.onresult = (event) => { let final = ''; for (let i = event.resultIndex; i < event.results.length; i++) { if (event.results[i].isFinal) { final += event.results[i][0].transcript; } } if (final) { document.getElementById('messageInput').value = final; // 自动发送 sendMessage(); } }; recognition.onerror = (event) => { console.error('语音识别错误:', event.error); }; recognition.start(); }
💡 浏览器兼容：Chrome、Edge、Safari 均支持。Firefox 部分支持。

10. 常见问题排查 第十步

问题可能原因解决方案
页面空白 config.js 未加载或 Token 无效 检查 config.js 是否存在，Token 是否正确
登录失败 用户名或密码错误 确认注册时保存的信息，检查 GitHub Issues 中的用户数据
消息发送失败 Token 权限不足或网络问题 确认 Token 有 repo 权限，检查控制台错误信息
部署后无法访问 Pages 未启用或路径错误 检查 Settings → Pages 是否启用，确认仓库名正确
AI 不回复 AppID 错误或网络问题 检查思知 AI AppID 是否正确，确认网络连接
语音识别不工作 浏览器不支持或未授权麦克风 使用 Chrome/Edge，允许麦克风权限
💡 调试技巧：按 F12 打开开发者工具，查看 Console 和 Network 标签页的错误信息。
